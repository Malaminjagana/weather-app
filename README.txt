# Weather App

A simple web application to check the current weather conditions of any city.

## Usage

1. Open the application in a web browser.
2. Enter the name of the city for which you want to check the weather.
3. Click the "Klick here" button.
4. The application will display the temperature and weather description for the entered city.

## How to Run Locally

To run the Weather App locally, follow these steps:

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser.
